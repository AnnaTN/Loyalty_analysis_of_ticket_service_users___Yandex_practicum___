# Анализ лояльности пользователей сервиса с билетами с помощью Python
Проект был выполнен в ходе обучения в Яндекс.Практикуме, профессия "Специалист по Data Science".

#### Краткое описание проекта
В работе использовалась выгрузка датасета, содержащая в себе данные о покупках пользователей сервиса билетов на различные мероприятия. Необходимо было проанализировать его данные, чтобы понять, какие пользователи с большей вероятностью возвращаются на платформу и делают заказы повторно. Выяснение этих данных позволяет быстро обнаруживать перспективных клиентов и предлагать им персонализированные условия, оптимизировать маркетинговые бюджеты и т.д.

#### Реализованные этапы проекта
- выгружен датасет с помощью SQL-запроса и подключения к базе данных;
- приведены значения выручки с заказов к единой валюте с помощью другого датасета;
- проведено ознакомление с исходным датасетом;
- проведена предобработка датасета;
- создан профиль пользователя;
- реализован исследовательский анализ данных;
- сделан вывод на основе проведенного анализа.

#### Результат анализа
- в наиболее лояльную группу входят пользователи, совершающие частые покупки, содержащие в среднем по 2-3 билета, к ним можно отнести семьи/группы знакомых (друзей). Для них можно более точно настроить рекламу основываясь на их предыдущих выборах, добавить скидки на билеты, бонусы за заказы;
- потенциально лояльные клиенты совершили больше одного заказа, количество билетов в их заказах близко к 2-3, как и у лояльной группы, заказы совершаются с меньшим промежутком времени между датами заказов. Для таких людей можно настроить персонализированные условия, информировать о новых мероприятиях;
- в наименее лояльную группы входят пользователи, посещающие мероприятия в одиночку (либо парами), большими группами, к ним можно отнести рабочие/студенческие/школьные коллективы. На них не оказывают влияние ни регион проведения мероприятия, ни его тип, ни вид устройства, ни оператор. Лояльность людей определяется скорее качеством первого опыта взаимодействия с сервисом и интересом к мероприятиям. Поэтому можно только предположить, что поможет повысить уровень лояльности подобных групп: например, привлечение большего их внимания к услугам сервиса, более частая реклама различных мероприятий, предложения о повторной покупке билетов, улучшение структуры сервиса с целью развития его визуальной привлекательности для появления большего желания совершения покупки.
